Install the MP3by4 Chrome Extension

Summarize webpages in one click while surfing the net! Load the developer build today.

  • 1

    Download Extension Files

    Locate the extension folder in the local project repository files on your machine.

  • 2

    Open Chrome Extension Page

    Type chrome://extensions/ in your Chrome address bar and hit enter.

  • 3

    Enable Developer Mode

    Toggle the "Developer mode" switch in the upper-right corner of the extensions page.

  • 4

    Load Unpacked Folder

    Click the "Load unpacked" button in the top-left and select the extension directory from our project folder.
